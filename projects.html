Projects

Language Lab

Language Lab is a language learning web application that allows users to practice the conjugations of the 25 most common verbs in English.

Book Fox

Book Fox is a book review web application where one can search for books, leave reviews for individual books, and see the reviews made by other people.

Finance

Finance is a business application via which you can manage portfolios of stocks, check stock prices, and buy/sell stocks by querying IEX for prices.

Flack

Flack is an online messaging service where one can send and receive messages in real time. Users can choose a display name, create channels to communicate in, as well as see and join existing channels.

Readability

Readability is a program built in C to compute the approximate (U.S.) grade level needed to comprehend a text using the Coleman-Liau index.

Substitution

Substitution Allows one to encrypt a message using a simple substitution cipher written in C.
