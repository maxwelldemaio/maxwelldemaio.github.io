Featured Projects

My best work.

LangLab

LangLab is a free and open source language learning website that allows users to practice the conjugations of the 25 most common verbs in a target language. Helpful resources have also been provided to help learners on their path.

Network

Network is a Twitter-esque social network website for making posts and following users built with Django and jQuery. Once signed in, one can create/like posts and follow other users for a more personalized social feed.

Flack

Flack is an online messaging service built with Flask-SocketIO where one can send and receive messages in real time. Users can choose a display name, create channels to communicate in, as well as see and join existing channels.

Wiki

Wiki is a Wikipedia-like online encyclopedia that allows users to view, create, and edit entries. When looking at an entry's page, the entry (stored in Markdown) is converted to HTML.

Book Fox

Book Fox is a book review web application where one can search for books, leave reviews for individual books, and see the reviews made by other people through the use of Goodreads' API.

Finance

Finance is a business application via which you can manage portfolios of stocks, check stock prices, and buy/sell stocks utlizing IEX's API. Also, with Chart.js users can view a pie chart of their portfolio distribution.
